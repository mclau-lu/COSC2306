# COSC2306
Course repository for COSC2306: The Unix/Linux Operating System course.

## Instructor

Dr. Meng Cheng Lau  
Bharti School of Engineering and Computer Science  
Laurentian University

## Course Description

This course introduces the Unix and Linux operating systems, including
command line usage, shell scripting, processes, file systems, and basic
system administration.

## Structure of this repository

- `weeks/`    weekly topics and lab notes  
- `assignments/`    assignment handouts and starter files  
- `scripts/`    demo scripts used in lectures  
- `resources/`    extra references and links  

## How students use this repository

1. Clone the repository  
2. Read the weekly notes under `weeks/`  
3. Complete assignments under `assignments/`  
4. Try the example scripts under `scripts/`
